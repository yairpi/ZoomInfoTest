# ZoomInfoTest
my App test for zoomInfo

### App running
In the home_test project directory, you can run:
#### `npm install`
#### `npx pod-install ios`
#### `npm start`

#### for run ios:
#### `npm run ios`

#### for run android:
#### `npm run android`

### Server running
In the sever project directory, you can run:

#### `npm install`
#### `npm run dev`

The Server run on port 3000
